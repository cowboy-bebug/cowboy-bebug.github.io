Prompt Engineering Tips

Posted at 2025-03-14 # ai # prompt # tips

Always Validate Prompts

I’ve run into this issue several times - updating prompts directly in the application doesn’t always result in updated responses. But when testing the same prompt in the Playground, the application suddenly starts reflecting the change.

It appears that something is being cached at the application layer, forcing us to use the Playground to trigger fresh responses.

TL;DR: When working on prompt engineering, always validate and iterate in the playground first.
